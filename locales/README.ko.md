# MyGPU: GPU 관리 도구

MyGPU는 NVIDIA `nvidia-smi`의 경량 버전으로, 우아한 웹 대시보드를 갖춘 컴팩트한 GPU 관리 유틸리티입니다.

## 기능

- **실시간 모니터링**: GPU 및 시스템 메트릭(사용률, VRAM, 전력, 온도)을 제공합니다.
- **관리 및 강제 실행**: VRAM 제한 설정, 프로세스 자동 종료(관리자 전용), 모니터링 목록 관리를 통해 GPU 리소스를 제어합니다.
- **벤치마킹 및 시뮬레이션**: GEMM 워크로드를 사용하여 시스템 안정성을 테스트하고, 3D 입자 물리 시뮬레이션을 통해 GPU 부하를 시각화합니다.

## 로드맵 및 미래 작업

기여를 환영합니다! 미래 개발 계획은 다음과 같습니다.

- **다중 GPU 지원**: NVLink 토폴로지를 포함한 다중 카드 시스템을 향상 지원합니다.
- **컨테이너화**: Docker 공식 지원을 통해 컨테이너 환경에서 쉽게 배포할 수 있습니다.
- **원격 액세스**: SSH 터널링 통합 및 보안 원격 관리를 구현합니다.
- **플랫폼 간 지원**:
  - **Linux 지원**: Ubuntu/Debian에 중점을 둡니다.
  - **Apple Silicon 지원**: macOS에서 Apple Silicon 모니터링을 지원합니다.
- **하드웨어 독립성**:
  - **AMD ROCm 지원**
  - **Intel Arc 지원**
- **다국어 문서화**: GitHub에서 가장 인기 있는 언어로 문서화를 지원합니다.

## 요구 사항

- **OS**: Windows 10/11
- **Python**: 3.10 이상
- **하드웨어**: NVIDIA GPU 및 설치 드라이버
- **CUDA**: 12.x 툴킷(벤치마킹/시뮬레이션 기능 사용 시 엄격히 필요)
  - *참고: CUDA 12.x가 감지되지 않으면 GPU 관련 벤치마킹 기능이 비활성화됩니다.*

## 설치

도구는 모듈식 설치를 지원하여 다양한 요구 사항에 맞게 설치할 수 있습니다.

### 1. 최소(CLI 전용)

헤드리스 서버나 백그라운드 모니터링에 적합합니다.

- 명령줄 인터페이스
- 기본 시스템/GPU 메트릭 제공

### 2. 표준(CLI + 웹 UI)

대부분의 사용자에게 적합합니다.

- 웹 대시보드 포함
- REST API 엔드포인트
- 실시간 차트
- 하지만 시뮬레이션이나 벤치마킹 기능은 없습니다.

### 3. 전체(표준 + 시각화)

개발 및 스트레스 테스트에 적합합니다.

- 시뮬레이션 기능 포함
- PyTorch/CuPy 의존성을 사용하여 벤치마킹 수행

### 빠른 시작

1. **다운로드** 최신 릴리스 또는 저장소를 클론합니다.
2. **설정 실행**:

   ```powershell
   .\setup.ps1
   ```

3. **시작**:

```powershell
# 웹 대시보드(표준/전체) 시작
python health_monitor.py web

# CLI 시작
python health_monitor.py cli
```
